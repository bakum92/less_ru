Это CSS, только немного больше

Используйте с Node.js:
npm install -g less
>lessc styles.less styles.css
Или браузер:
Для учебы
<link rel="stylesheet/less" type="text/css" href="styles.less">
<script src="//bakum92.github.io/less/less.min.js"></script>
<link rel="stylesheet/lessx" type="text/css" href="styles.lessx">
Для профессиональная
<link rel="stylesheet" href="styles.less" type="text/less"

Less (что означает Leaner Style Sheets) является обратно совместимым языковым расширением для CSS. Это официальная документация для Less, языка и Less.js, инструмента JavaScript, который преобразует ваши стили Less в стили CSS.

Поскольку Less выглядит точно так же, как CSS, его изучение очень просто. Less делает лишь несколько удобных дополнений к языку CSS, что является одной из причин, по которой его можно выучить так быстро.

  • переменные
  • Примеси
  • гнездование
  • Вложенные At-Rules и Bubbling
